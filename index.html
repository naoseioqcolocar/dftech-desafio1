Fundamentos de hardware

  1. Definição
    • Hardware é uma palavra de origem inglesa que, no âmbito da informática, é utilizada para designar a parte física de um computador.
  2. Evolução
    • Hardware sofreu muitas mudanças nos últimos 13 anos, aumentou velocidade, diminui tamanho, aumentou qualidade de vídeo, se tornou fácil de ser transportado e também houve mudanças no material de fabricação.
  3. Arquitetura (componentes e periféricos)
    • A arquitetura de computadores é a forma como os diversos componentes de um computador são organizados, determina aspectos relacionados à qualidade, ao desempenho e à aplicação para a qual o dispositivo vai ser orientado
  4. Funcionamento
    • os hardwares se conectam para levar energia à máquina, possibilitar imagens nítidas na tela e, sobretudo, permitir a atuação dos softwares.

Fundamentos de Software

  1. Definição
    • Um software é um serviço computacional utilizado para realizar ações nos sistemas de computadores.
  2. Evolução
    • A evolução do software é referida como o processo de desenvolvimento, manutenção e atualização do software por vários motivos.
  3. Tipos e caracteristicas
    • Software de base: sistemas operacionais;
      Software de aplicativo: aplicações para atender os usuários finais;
      Software cliente-servidor: aplicação para ser utilizado por uma rede ou um servidor.

Fundamentos de Protocolos

  1. Definição
    • protocolo é uma convenção que controla e possibilita uma conexão, comunicação, transferência de dados entre dois sistemas computacionais.
  2. Evolução
    • Software de base: sistemas operacionais;
      Software de aplicativo: aplicações para atender os usuários finais;
      Software cliente-servidor: aplicação para ser utilizado por uma rede ou um servidor.
  3. Tipos e características (classificação, estrutura e modelos)
    • HTPPS
      • HTTPS é um protocolo que permite que o computador troque informações com o servidor que abriga o site, deixando-o mais seguro.
    • FTP
      • FTP (File Transfer Protocol) é um protocolo de rede para a transmissão de arquivos entre computadores
    • SMTP
      • SMTP (Protocolo de Transferência de Correio Simples, ou “Simple Mail Transfer Protocol”, em inglês) é um protocolo de comunicação, ou conjunto de regras usado para enviar e-mail de um cliente de e-mail ou provedor de webmail para o servidor de e-mail de um destinatário

    Categorias de Software

    • Categorias de Software
    • Conceitos
    • Aplicações
    Categorias de software se referem a diferentes tipos de programas de computador que são projetados para desempenhar funções específicas. As principais categorias incluem:
    Software: conjunto de programas que controlam um dispositivo eletrônico. Sistema Operacional: software que gerencia recursos de hardware e software. Aplicativo: software que executa uma tarefa específica para o usuário. Firmware: software embutido em dispositivos eletrônicos. Biblioteca de Software: coleção de código pré-escrito para desenvolver programas. Open Source: software com código aberto para visualização, modificação e distribuição.
    Aplicações de software são programas de computador projetados para realizar tarefas específicas, como processamento de texto, edição de imagens, gerenciamento de banco de dados ou navegação na web. Eles são desenvolvidos para serem executados em sistemas operacionais específicos e podem ser instalados em dispositivos como computadores pessoais, smartphones e tablets.
